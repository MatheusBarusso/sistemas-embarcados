# sistemas-embarcados
Repositório direcionado para o projeto da disciplina de Sistemas Embarcados


## NES 
- Latch -> Preto
- Data -> Verde
- Pulse -> Azul 
- VCC -> Vermelho
- GND -> Amarelo

## Genesis 6 button
- UP -> 1 -> Vermelho
- DOWN -> 2 -> Preto
- LEFT -> 3 -> Cinza
- RIGHT -> 4 -> Laranja
- VCC -> 5 -> Marrom
- B -> 6 -> Verde
- SELECT -> 7 -> Branco
- GND -> 8 -> Azul
- C -> 9 -> Amarelo

## Gamecube Controller
- DATA -> Vermelho
- VCC -> Preto
- GND -> Marrom
